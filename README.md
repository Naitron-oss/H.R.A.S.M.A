# Chat-Appication-using-Cpp-
A console based chat application using C++.

Features:
 * Sign up
 * Sign in
 *	Inbox
 *	Chat
 *	Friend list
 *	Add friend
 *	Unfriend
 *	Group Message
 

Steps to execute.
  * Compile the file on any GNU g++ compiler.
  * Register on the application following instructions displayed on console.
  * Login and Enjoy!
